## Install

The **recommended** way to install PostDoc is by running:

```bash
npm init postdoc@latest
```

Or you can add PostDoc to an existing project manually: 

```bash
npm i -D postdoc
```

and then run `npx postdoc init` to create a new PostDoc project.
